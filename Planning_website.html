This version is split into two different tables. These two tables are the series table so that is all the information from the series including IMBD or age rating ect. The actor table incldes all the information about the actor. The purpose of this version to be able to make a database and be able to add new queries to it. I want to be able to get informaion across the two tables. This is so that if actors for example Zendaya where she is in more than one TV show then her information doesn't have to be repeated.
